Заключение:

1. Возможность проинициализировать сразу нескоьлко переменных.

2. Списки - переменные, которые могут хранить множественные элементы данных в
пронумерованных элементах, индекс которых начинается с нуля.

3. К данным, хранящимся в элементе списка можно обратиться, используя имя списка
и индекс элемента в квадратных скобках.

4. Функция len() возвращает длину указанного списка.

5. Кортеж в Python - неизменяемый список, элементы которого можно присвоить отдельным
переменным с помощью так называемой распаковки последовательности.

6. К данным, хранящимся в элементе кортежа, можно обращаться, используя имя кортежа и
номер индекса в квадратных скобках.

7. Множество в Python - упорядоченный набор уникальных элементов, значения которых
могут быть обработаны с помощью методов. К данным, хранящимся в множестве, нельзя
обращаться по индексу элемента.

8. Словарь в Python представляет собой список пар ключ:значение, в которых каждый
должен быть уникален.

9. К данным, хранящемся в элементе словаря, можно обращаться, используя имя словаря и
ключ в квадрытных скобках.

10. Ключевое слово if  осуществляет проверку условного выражения на равенство логическим
значениям True и False.

11. С помощью for in можно перебирать элементы указанного списка, либо строки.

12. Функция range() генерирует числовую последовательность, которую можно использовать для
указания длины цикла for in.

13. Ключевые слова break и continue используются для прерывания итераций цикла.
